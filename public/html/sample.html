    Data structure where pairs of parents are linked to children

    I'm looking to make something where each parent, although items that can be made individually, must be linked with some other specific element to create a child node.

    For example, you could have something like

     A + B    C + D    C + G
  \ /      \ /      \ /
   E        E        H
              \    /
                I

    But B + C do not make E in this example, because only specific pairs of parents point towards a child node.

    This is distinct from a directed graph in that it is not simply two elements that point towards another, but a specific pair of elements that point towards another. What is a name for this, and how could it be programmed (in Python)?
